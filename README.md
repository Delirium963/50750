Trabajo de Analizador UTN FRM
USO DEL ANALIZADOR

 1. Ingresar al documento 'input.txt' el mensaje a analizar
 2. Guardar el documento 'input.txt'. ESTO ES IMPORTANTE
 3. Abrir el terminal de la pc (Win + R, ingresar el comando cmd y pulsar Enter).
 4. Ingresar el comando "cd" seguido de la ubicación donde se descargó el analizador
 (Ejemplo: cd C:\Escritorio\Analizador)
 5. Ingresar el comando "node Analizador.js"
 6. Esperar que el analizador termine de ejecutarse 
 7. Y ya tenés los resultados
 8. Repetir los pasos 1 y 2 para un nuevo texto a analizar y el paso 5 para obtener un nuevo analisis
